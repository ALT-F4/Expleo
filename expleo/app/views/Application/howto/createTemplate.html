How to create templates

You can create templates by using plain text, latex or docx files. Terms which should be replaced in the template should be placed into %% tags. (eg. %%xyz%%, %%abc:xyz%%) If you want to have categories use the example "%%abc:xyz%%".

Example upload:

	Hello %%Receiver:First name%% %%Receiver:Last name%%!

	regards %%Sender%%
